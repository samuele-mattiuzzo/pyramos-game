Classes for handling high score tables.

High

Create a Highs object and returns the default high score table. Arguments: fname -- filename to store high scores in limit -- limit of scores to be recorded, defaults to 10

_High

A high score table. These objects are passed to the user, but should not be created directly. You can iterate them: for e in myhigh: print e.score,e.name,e.data You can modify them: myhigh[0].name = 'Cuzco' You can find out their length: print len(myhigh)

_High.save

Save the high scores.

_High.submit

Submit a high score to this table. Return -- the position in the table that the score attained. None if the score did not attain a position in the table.

_High.check

Check if a score will attain a position in the table. Return -- the position the score will attain, else None

Highs

The high score object. Arguments: fname -- filename to store high scores in limit -- limit of scores to be recorded, defaults to 10 You may access _High objects through this object: my_easy_hs = highs['easy'] my_hard_hs = highs['hard']

Highs.load

Re-load the high scores.

Highs.save

Save the high scores.
